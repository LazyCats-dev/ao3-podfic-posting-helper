Title template

Set the title template for your work. ${title} will be replaced with the original title, while ${author} or ${authors} will be replaced with the author's name, or a comma-separated list of names if there are multiple authors. ${author} and ${authors} both do the same thing, this is just to reduce typo-related frustration.

For example, if the original work is called "Super awesome title" and is by "author1" and "author2":

  • "${title}" will give you "Super awesome title"
  • "[Podfic] ${title}" will give you "[Podfic] Super awesome title"
  • "podfic of ${title} by ${authors}" will give you "podfic of Super awesome title by author1, author2"

Summary template

Set the summary template for your work. In particular, this allows the following replacements:

  • ${blocksummary} will be replaced with the original summary, wrapped in a blockquote. Because of the way ao3 handles block quotes, you should put whatever you want to follow this on the same line.
  • ${summary} will be replaced with the original summary.
  • ${title} will be replaced with the original title. This will link to the original work.
  • ${authors} will be replaced with the author's name, which will link to their AO3 page. If there are multiple authors, this will be a comma-separated list of authors.
  • ${author} does the same as ${authors}, to reduce typo-related frustration.

The original version of this extension allowed you to pick between two default options, "${summary}" and "${blocksummary}Podfic of ${title} by ${author}.".
